# word-guessing-game
A beginner-friendly word guessing game created as my first BCA project. It demonstrates basic logic, loops, and user input handling.I built this project to practice and learn basic programming concepts.

## Features
-Random word selection
-User input guessing
-Contains limited attempts
-Simple console-based game

## Technologies used
-Programming language- Python

## Editor- VS Code

## Learnings
-Understanding of loops
-Basic game logic
-Conditional statements
